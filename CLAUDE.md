> **项目名称**: 《非遗新生》/《非遗传承》

> **作者**: 葛露怿
> **创建日期**: 2025-2026
> **最后更新**: 2026-04-17

---

## 1. 项目概述

本项目是一个轻量级 Web 前端单页应用（SPA），旨在通过数字化技术（3D 展示、AR 体验、虚拟工坊等）展示和推广中国非物质文化遗产。目标受众为大学生和年轻群体，涵盖剪纸、皮影戏、瓷器制作、刺绣、年画等传统手工艺的数字传承。

### 1.1 核心定位

- **文化定位**: 非遗文化 × 数字科技的跨界融合
- **技术定位**: 纯前端 SPA，无需后端，可直接通过浏览器或 Live Server 运行
- **竞赛定位**: 计算机设计大赛国赛作品，面向大一学生可展示复杂度

## 4. 架构设计

### 4.1 整体架构模式

项目采用 **单页应用（SPA）+ Vue.js 响应式路由** 架构：

```
┌─────────────────────────────────────────────────┐
│                   HTML 文件                      │
│  ┌───────────┐  ┌─────────────┐  ┌────────────┐ │
│  │ 内嵌 CSS   │  │  Vue 模板   │  │ Vue 逻辑   │ │
│  │ 样式定义   │  │  12个页面    │  │ 路由/状态   │ │
│  └───────────┘  └─────────────┘  └────────────┘ │
└─────────────────────────────────────────────────┘
         │                    │
    CSS class 切换      Vue reactive state
    .page / .active     currentPage 变量
```

**页面切换机制**: 所有页面共存于一个 HTML 文件中，通过 Vue 的 `currentPage` 响应式变量控制 `.page` / `.page.active` CSS 类（`display: none/block`）实现切换，无需前端路由库。

### 4.2 12 个页面模块

#### 前端用户侧（9 页）

| # | 页面 | 功能描述 |
|---|------|----------|
| 1 | 首页门户 | 品牌展示、非遗类别导航、热门展品推荐、数据看板 |
| 2 | 分类展示页 | 按手工艺类别浏览展品（剪纸、皮影、瓷器、刺绣等） |
| 3 | 详情页 | 单个非遗展品的图文详情、历史渊源、制作工艺 |
| 4 | 3D 沉浸展示 | 使用 Three.js 渲染 3D 文物模型，可旋转/缩放交互 |
| 5 | WebAR 体验 | 基于浏览器的 AR 体验，手机扫描查看虚拟文物 |
| 6 | 虚拟工坊首页 | 手工艺制作虚拟工坊入口 |
| 7 | 分步制作交互 | 引导用户分步体验非遗制作流程 |
| 8 | 作品分享页 | 用户生成作品的展示、点赞、评论、分享 |
| 9 | 问答页 | 非遗知识问答/Trivia，增加互动趣味性 |

#### 后台管理侧（3 页）

| # | 页面 | 功能描述 |
|---|------|----------|
| 10 | 管理员登录 | 后台管理登录入口 |
| 11 | 内容管理列表 | 展品内容的增删改查 |
| 12 | 资源上传 | 图片、视频等素材上传管理 |

## 5. 设计规范

### 5.1 配色方案

| 角色 | 色值 | 用途 |
|------|------|------|
| 主色 | `#C41E3A` | 中国红，按钮、标题、强调元素 |
| 深色主色 | `#8B1538` | 深红，hover 状态、渐变 |
| 辅助色 | `#F4D03F` | 金色，图标、评分、装饰 |
| 背景色 | `#f5f0e6` / `#FAF3E8` | 暖色调纸张/奶油色背景 |
| 深色背景 | `#1a1a2e` | 深蓝夜空，沉浸式页面背景 |

### 5.2 CSS 动画

项目使用 CSS Keyframe 动画实现丰富的过渡效果：

- `fade-in` — 淡入效果
- `slide-in` — 滑入效果
- `scale-in` — 缩放进入
- 鼠标悬停阴影效果
- 渐变背景动画

### 5.3 字体

- feiyi_new 版本使用 Google Fonts 的 **Noto Serif SC**（思源宋体），契合非遗文化的传统感
- 主版本使用系统默认中文字体

## 6. 开发工作流

### 6.1 开发服务器

通过 VS Code Live Server 扩展运行，端口固定为 **5501**。

### 6.2 Git 提交规范

项目采用简单 Git 工作流，注意事项记录在文件 `每次提交记得在最新版提交...` 中：
- **必须在最新提交基础上继续提交**，避免分支分叉
- 建议提交前 pull 最新代码

### 6.3 资源管理

- 图片存放于 `images/` 目录，命名规则为数字序号或描述性名称
- 视频存放于 `videos/` 目录，拼音缩写命名（ci=瓷, jian=剪, pi=皮 等）
- 大体积媒体文件（5-43MB）已就绪，可直接在 HTML 中引用
