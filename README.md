# 非遗文化数字化展示平台 - 《非遗新生》

## 🎯 项目简介

本项目是第十七届中国大学生计算机设计大赛参赛作品，旨在通过数字化技术（3D 展示、AR 体验、虚拟工坊等）展示和推广中国非物质文化遗产。项目采用纯前端 SPA + Node.js 后端架构，轻量级、易部署，适合比赛演示。

## 🚀 快速开始

### 1️⃣ 启动后端服务器

```bash
cd backend
npm install          # 首次安装依赖（约1分钟）
node init-data.js    # 初始化数据库（仅首次需要）
node server.js       # 启动服务器
```

✅ 服务器运行在 `http://localhost:3000`  
✅ 管理员账号: `admin` / `admin123`

### 2️⃣ 访问前端

- **方式1**: 直接用浏览器打开 `插入图片视频后.html`
- **方式2**: 使用 VS Code Live Server（推荐）
  - 在文件上右键 → "Open with Live Server"
  - 访问 `http://localhost:5501/插入图片视频后.html`

### 3️⃣ 测试功能

1. **浏览展品**: 首页 → 分类展馆 → 点击展品查看详情
2. **管理员登录**: 点击导航栏"后台管理" → 输入账号密码
3. **管理展品**: 登录后可增删改查展品内容

## 🔧 核心功能

### 前端（插入图片视频后.html）
- ✅ **12 个页面**: 首页、分类展馆、详情页、3D 展示、AR 体验、虚拟工坊、答题互动、后台管理等
- ✅ **Vue.js 3 响应式路由**: 单页应用，页面切换流畅
- ✅ **真实媒体资源**: 已集成图片和视频
- ✅ **API 数据绑定**: 从后端动态加载展品数据

### 后端（backend/server.js）
- ✅ **RESTful API**: 展品管理、用户数据、文件上传
- ✅ **JWT 认证**: 管理员登录和权限验证
- ✅ **JSON 存储**: 轻量级，无需数据库安装
- ✅ **文件上传**: 支持图片、视频、3D 模型上传

## 📊 数据库设计

`backend/data/database.json` 包含 4 个数据集合：

1. **admins** - 管理员账号（BCrypt 哈希密码）
2. **exhibits** - 非遗展品（标题、分类、图片、视频、历史、工艺等）
3. **favorites** - 用户收藏记录
4. **view_history** - 浏览历史记录

## 🏆 比赛演示要点

1. **数据持久化**: 后端保存展品数据，刷新不丢失
2. **管理功能**: 管理员可登录并管理展品内容
3. **用户交互**: 收藏、浏览历史、答题等互动功能
4. **完整闭环**: 从展示 → 交互 → 分享的完整体验
5. **轻量部署**: 无需复杂环境，直接运行即可演示
